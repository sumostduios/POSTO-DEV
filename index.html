Oyun haqqında

POSTO — sürətli top-down 2D oyun. Müxtəlif səviyyələr, multiplayer/yerli modüllər, və sadə nəzarət ilə oynanır. Bu səhifə yalnız Azərbaycan dilində təqdim olunur.

Janr: Top-down / Aksiya
Studiya: Sumo Studios
Platforma: Windows (yaxın gələcəkdə daha çox)

Xüsusiyyətlər

  • Çox oyunçu dəstəyi (yerli və onlayn).
  • Sürətli oynanış və müxtəlif silahlar.
  • Hafif sistem tələbləri — 128–512 MB RAM üçün optimallaşdırma məqsədləri.
  • Sadə və intuitiv nəzarət.

Yeniliklər

Son yeniliklər və yamaq qeydləri üçün bu səhifəni izləyin.

Sistem tələbləri

Minimum

  • OS: Windows 7/8/10 (32/64 bit)
  • RAM: 128 MB (ən azı) — oyunun yüngül versiyası üçün
  • Prosessor: 1.2 GHz və ya ekvivalent
  • Disk: 50 MB boş yer

Tövsiyə olunan

  • OS: Windows 10
  • RAM: 512 MB
  • Prosessor: 1.6 GHz və ya daha yaxşı
  • Disk: 100 MB boş yer
